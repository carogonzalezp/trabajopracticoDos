Bootstrap

Qué es?

Es el Kit de herramientas de código abierto front-end, que presenta variables, sistemas de grillas, componentes prediseñados y complmentos de Javascript. Además tiene una serie de estilos para crear columnas fácilmente.

Tipos de contenido

  • Tipografí
  • imágenes
  • tablas
  • navegación,etc.

También se pueden encontrar distintas Utilidades:

  • Utilizar Bordes
  • Colores
  • flex
  • float
